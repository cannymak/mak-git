Brief Idea about the Project:

Input: Hospital data set (Csv files, MySQL Db files, Streaming Data)
Value: Locating the nearby speciality hospitals in the range of 100 meters
Output: Shorterm(Reports on dataset, processing, and outcome) and longterm(Mobile App)

Technologies Used:
Hadoop, HDFS, MapReduce, Pig, Hive, Sqoop, Flume, QlikView

Future technologies to Use:
Solr, Spark, HBase, R, Tableau
